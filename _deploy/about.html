How I do my computing

(Title stolen from Richard Stallman). Software I use is all free (as in freedom) and open source software (with an exception mentioned below). I write all my texts, presentations, code in plain text and convert the source text into format required.

I use:

  • linux operating system
  • markdown for typesetting using any text editor (notepad)
  • pandoc for converting the plain text into any format (.docx,.pdf,.ppt,.html,.epub,..)
  • R for data analysis, statistical analysis & graphics
  • .git for version control

The only non-free software I use is BitTorrent Sync that I use for syncing my workspaces between my devices so, that my files newer reside on any third party server.
