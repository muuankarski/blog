IndicatorOecdR Documentation

List the currently available indicators from OECD Country statistical profile

Description

List the currently available indicators from OECD Country statistical profile at: http://www.oecd-ilibrary.org/economics/country-statistical-profile-russian-federation_20752288-table-rus

Usage

  IndicatorOecd(urli = "http://research.muuankarski.org/rustfare/data/IndicatorOecd.RData")

Arguments

urli

is the url address to the data

Value

data frame with indicator names

Author(s)

Markus Kainu <markuskainu(at)gmail.com>

Examples

# available.indicators <- IndicatorOecd()
