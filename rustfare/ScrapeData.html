ScrapeDataR Documentation

Scrape the raw html-table data

Description

Scrape the raw html-table data from Rosstat Regional Statistics

Usage

  ScrapeData(indicator = "mortality_rate")

Arguments

indicator

Character string. Select indicator from the indicator column of data frame returned by function RosstatIndicator.

Value

data frame with raw data from html-table
