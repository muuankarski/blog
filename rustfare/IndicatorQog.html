IndicatorQogR Documentation

List the currently available indicators from Quality of Governance database

Description

List the currently available indicators from Quality of Governance database http://www.qog.pol.gu.se/data/datadownloads/qogstandarddata/

Usage

  IndicatorQog(urli = "http://www.qog.pol.gu.se/digitalAssets/1358/1358067_qog_tsd_csv_v6apr11.csv")

Arguments

urli

is the url address to the data

Value

string of names of available indicators
