IndicatorRosstatR Documentation

List the currently available indicators from Rosstat Regional Statistics

Description

RosstatIndicator loads data frame that lists currently available indicators from Rosstat Regional Statistics

Usage

  IndicatorRosstat(urli = "http://research.muuankarski.org/rustfare/data/RosstatIndicatorData.RData")

Arguments

urli

is the url address to the data

Value

data frame with indicator name, url, levels of aggregation and metadata in Russian and in English

Author(s)

Markus Kainu <markuskainu(at)gmail.com>

Examples

# available.indicators <- IndicatorRosstat()
