rustfareR Documentation

rustfare: toolbox for R language for open welfare data from Russian Federation

Description

The rustfare package pools together tools for searching, extracting, and formatting welfare data on Russian Federation from different data providers. The rustfare functions are named according to data provider or the data set. Currently there data available from: Rosstat Regional Statistics, OECD, Quality of Governance -institute and World Bank
