ScrapeData_skip1R Documentation

Scrape the raw html-table data (skipping the 1st row)

Description

Scrape the raw html-table data from Rosstat Regional Statistics for tables with rowspan used on first row.

Usage

  ScrapeData_skip1(indicator)

Arguments

indicator

Character string. Select indicator from the indicator column of data frame returned by function RosstatIndicator.

Value

data frame with raw data from html-table

Author(s)

Markus Kainu <markuskainu(at)gmail.com>

Examples

# raw.html.data <- ScrapeData_skip1("")
