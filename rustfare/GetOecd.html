GetOecdR Documentation

Download data from OECD Country statistical profile: Russian Federation

Description

Download data from OECD Country statistical profile: Russian Federation http://www.oecd-ilibrary.org/economics/country-statistical-profile-russian-federation_20752288-table-rus

Usage

  GetOecd(measure = "public_expenditure_on_health")

Arguments

indicator

Character string. Select indicator from the indicator column of data frame returned by function OecdIndicator. More than one indicator is NOT allowed.

Value

data frame with 4 columns

  • indicator Name of the indicator

  • unit unit the indicator

  • value Value of the indicator

  • year Year of observation
